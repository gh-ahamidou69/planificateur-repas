Planificateur de repas de la semaine

Ce projet est un planificateur de repas interactif pour toute la semaine, développé en HTML, CSS et JavaScript. Il permet de saisir ses repas ou de générer automatiquement des suggestions pour chaque jour.

Fonctionnalités

Saisie manuelle des repas pour chaque jour de la semaine (Petit-déjeuner, Déjeuner, Dîner).

Sauvegarde automatique dans le LocalStorage pour conserver les données même après fermeture du navigateur.

Génération automatique de repas aléatoires pour toute la semaine.

Réinitialisation facile des repas via un bouton.

Interface simple et responsive.

Technologies utilisées

HTML5

CSS3

JavaScript (ES6+)

Installation / Utilisation

Cloner le dépôt :

git clone https://github.com/<votre-utilisateur>/planificateur-repas.git

Placer les fichiers dans un serveur local (XAMPP, Laragon, ou utiliser VSCode Live Server).

Ouvrir index.html dans votre navigateur.

Utilisation :

Saisir vos repas dans les champs correspondants.

Cliquer sur Générer repas aléatoires pour remplir automatiquement.

Cliquer sur Réinitialiser la semaine pour tout vider.
